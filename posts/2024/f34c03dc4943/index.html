  • -3 * 10^4 <= nums[i] <= 3 * 10^4
  • 除了某个元素只出现一次以外，其余每个元素均出现两次。

首先介绍位运算中”异或运算“的三个性质

  • 任何数和 0 做异或运算，其结果都是它本身。即a⊕0=aa⊕0=aa⊕0=a
  • 任何数和它本身做异或运算，其结果是0。即a⊕a=0a⊕a = 0a⊕a=0
  • 异或运算满足交换律和结合律，即a⊕b⊕a=b⊕a⊕a=b⊕(a⊕a)=b⊕0=ba⊕b⊕a=b⊕a⊕a=b⊕(a⊕a)=b⊕0 = ba⊕b⊕a=b⊕a⊕a=b⊕(a⊕a)=b⊕0=b

根据本题给出的一个提示：除了某个元素只出现1次，其他元素均出现了两次。那么根据上面提到的三个性质，我们可知：除了我们要找出来的元素，其他的元素全部做异或运算，结果是 0，因为除了我们要找出来的元素，其他元素都出现了两次；而 0 和 一个非 0 值做异或运算，结果是原来这个非 0 值本身

  • -2^31 <= nums[i] <= 2^31 - 1
  • nums 中，除某个元素仅出现 一次 外，其余每个元素都恰出现 三次

这道题和上一题的区别在于重复元素的个数——重复元素的个数不再是2个，而是3个。这样的话就不能再用上面提到的异或运算来判断了

因为数组中的元素都在 int 范围内，所以可以依次计算要找出的元素的每个二进制位是 0 还是 1

具体来讲，考虑要找出元素的第iii 个二进制位（iii 从 0 开始编号），它可能是 0 或者 1。对于数组中非答案的元素， 每一个元素都出现了 3 次， 对应着 第iii 个二进制位的 3 个 0 或者 3 个 1， 无论是哪种情况，它们的和都是 3 的倍数。因此，要找出的元素的第iii 个二进制位就是数组中所有元素的第iii 个二进制位之和除以 3 的余数

这样的话，对于数组中的每一个元素xxx ，使用位运算(x>>i)&i(x >> i) \& i(x>>i)&i 得到xxx 的第iii 个二进制位，并将它们相加再对 3 取余，得到的结果一定是 0 或 1，即为要找出元素的第iii 个二进制位
